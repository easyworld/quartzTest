一个基于quartz的web url定时调度器
=================

功能
-------------------

* 使用cron表达式访问某url
* 支持选择GET方式与POST方式请求
* 支持调用后查看日志
* 支持任务暂停

待改进的地方
-------------------

1. 不支持https
2. 主页面需要使用ajax定时刷新
3. 有待排除一些异常bug

涉及技术框架
-------------------

* jQuery 
* bootstrap
* JSP
* SpringMVC
* quartz
* maven
